Den här appen andvänder:

-React
-Firebase

Syfftet med appen är att den ska effiktivera arbetsättet med Seo analysen. Man fyller i formulären med värden som skickas till databasen(firebase). Sen läser man av de värden och multiplicerar de för att få en snitt värde som man betygsätter sidan man analyserade. 

Stylat för att spara som pdf dokument som ska skickas till kunden.
